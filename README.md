# sambrit.com

Ground-up rebuild of the personal website for Sambrit Adhikari.

## Project shape

- `index.html`: full single-page site (HTML, CSS, JS)
- `CNAME`: GitHub Pages custom domain
- `robots.txt`: crawler rules
- `sitemap.xml`: search discovery

## Design goals

- Executive-level visual presence
- Low-noise, high-trust tone
- Subtle personality and interactivity
- Fast loading and mobile-friendly layout

## Local run

```bash
python3 -m http.server 8000
```

Open `http://localhost:8000`.
